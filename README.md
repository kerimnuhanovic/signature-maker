# Signature Maker

## How to Run the Project

Follow these steps to get the project up and running locally:

1. **Clone the Project**
   ```bash
   git clone git@github.com:kerimnuhanovic/signature-maker.git

2. **Navigate to project directory**
    ```bash
    cd signature-maker

3. **Install packeges**
    ```bash
    npm install
    
4. **Staart development server**
    ```bash
    npm run dev
